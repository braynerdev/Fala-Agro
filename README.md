# **Fala Agro - Desenvolvimento Front-End**

Este repositório contém o desenvolvimento web do front-end do site **Fala Agro**, um projeto criado por alunos da **Faculdade Senac** com o objetivo de solucionar um desafio da empresa **Fala Agro**.
